<h1 align="center"> 로지스틱 회귀
모델을 사용하여 타이타닉 데이터셋에서 승객의 생존 여부를 예측하는 모델을 만들었다.  </h1>
이 모델은 이진 분류 문제를 해결하는 데 사용됩니다. 구체적으로, 타이타닉 데이터셋에서 승객의 특성을 기반으로 그 승객이 생존했는지(1), 사망했는지(0)를 예측한다.
